
# Inversiones B/L - Sitio estático

Este repositorio contiene una página HTML que replica el diseño de la
captura que compartiste: un header azul, tarjeta principal, acciones,
registro de inspección, lista de envíos, servicios y datos de contacto.
Todas las secciones se construyen con Tailwind CSS (CDN) y no requieren
ningún proceso de construcción.

## Cómo verlo

1. Abre el directorio en Visual Studio Code.
2. Instala la extensión **Live Server** (o usa cualquier servidor estático).
3. Haz clic derecho en `index.html` y selecciona **Open with Live Server**.
4. Navega a `http://127.0.0.1:5500` o el puerto que Live Server asigne.

El archivo `index.html` ya contiene todos los estilos y scripts necesarios.
Puedes editar el contenido directamente para cambiar textos, enlaces o
colocar tus propias imágenes en `public/images`.
